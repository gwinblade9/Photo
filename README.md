# 📸 Photography Portfolio Website

Премиальный сайт-портфолио для фотографа с полным функционалом: галерея, блог, админ-панель, тёмная тема и многое другое.

## ✨ Особенности

### 🎨 Дизайн и UX
- **Минималистичный дизайн** с акцентом на фотографии
- **Адаптивный интерфейс** — идеально работает на всех устройствах
- **Тёмная/светлая тема** с сохранением выбора пользователя
- **Плавные анимации** при загрузке и наведении
- **Lightbox** для просмотра фото с навигацией и скачиванием

### 📁 Галерея
- Сетка фотографий с соотношением сторон 4:5
- Фильтрация по категориям (Свадьбы, Портреты, Природа)
- Пагинация и ленивая загрузка
- Водяной знак на preview-изображениях

### ✍️ Блог
- Создание и редактирование постов
- Комментарии от посетителей
- Счётчик просмотров
- HTML-форматирование контента

### 💼 Услуги и цены
- Управление пакетами услуг через админку
- Отображение цен и особенностей
- Кнопка связи с фотографом

### 📞 Контакты
- Форма обратной связи
- Сохранение сообщений в localStorage
- Отметка о прочтении в админке

### 🔧 Админ-панель
- Защищённый вход (admin@photographer.com / admin123)
- CRUD операции для фотографий
- Управление блогом
- Просмотр и управление сообщениями
- Настройка услуг
- Переключение темы

### 🎯 Уникальные фишки
- **Интерактивный слайдер "До/После"** для демонстрации ретуши
- **Плавный скролл** при навигации
- **Кнопка "Наверх"** с анимацией
- **Автосохранение** всех данных в localStorage

🐛 Устранение неполадок
Проблема: Стили не применяются
Решение: Убедитесь, что Tailwind CSS установлен:

bash
npm list tailwindcss
Проблема: Фото не загружаются
Решение: Проверьте, что URL изображения доступен. Для локальных фото положите их в public/images/.

Проблема: Не сохраняются данные после перезагрузки
Решение: Данные хранятся в localStorage. Очистка кэша браузера удалит их.

Проблема: Ошибка при запуске
Решение: Удалите node_modules и переустановите:

bash
rm -rf node_modules package-lock.json
npm install
npm run dev
Проблема: Белый экран
Решение: Откройте консоль браузера (F12) и посмотрите ошибки. Чаще всего проблема в неправильном импорте компонента.
