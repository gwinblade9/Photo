# Фото-портфолио

Сайт-портфолио для фотографа. Адаптивный, с тёмной темой, админ-панелью, галереей, блогом и формой обратной связи. Работает с бэкендом на Node.js + SQLite.

## Что внутри

- Галерея с фильтром по категориям (свадьбы, портреты, природа)
- Lightbox для фото (стрелки, описание, кнопка скачать)
- Блог с комментариями от посетителей
- Услуги и цены (можно добавлять и менять через админку)
- Контактная форма — сообщения сохраняются в базе данных
- Админ-панель: управление фото, блогом, услугами, просмотр сообщений
- Тёмная / светлая тема (переключается везде, включая админку)
- Слайдер "до/после" на главной
- Адаптив под телефоны, планшеты и компьютеры
- Загрузка фото и изображений для блога с компьютера

## Технологии

**Фронтенд:**
- React + Vite
- React Router
- Tailwind CSS
- Framer Motion

**Бэкенд:**
- Node.js + Express
- SQLite (файловая база данных)
- JWT для авторизации
- Multer для загрузки файлов

API эндпоинты (бэкенд)
Метод	URL	Описание
GET	/api/health	Проверка работы
POST	/api/auth/login	Вход в админку
GET	/api/photos	Список фото
POST	/api/photos	Добавить фото (только админ)
DELETE	/api/photos/:id	Удалить фото (только админ)
GET	/api/blog	Список статей
GET	/api/blog/:slug	Одна статья
POST	/api/blog	Создать статью (только админ)
PUT	/api/blog/:id	Обновить статью (только админ)
DELETE	/api/blog/:id	Удалить статью (только админ)
POST	/api/blog/:postId/comments	Добавить комментарий
GET	/api/services/public	Список услуг
POST	/api/messages	Отправить сообщение
